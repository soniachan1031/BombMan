# 💣 BombMan – MonoGame Bomb Strategy Game

Welcome to **BombMan** – our first mini-game built using **C# with MonoGame**, inspired by the classic Bomberman style!

## 🎮 Game Overview

**BombMan** is a strategic bomb-placement game where players must outsmart enemies, navigate breakable obstacles, and collect power-ups to survive. Developed as a fun and challenging project, the game is available on both **desktop** and **Android** platforms.

---

## 🧩 Key Features

- **💣 Strategic Bomb Placement**: Bombs explode in a cross pattern after a short delay—timing is key!
- **🧱 Dynamic Levels**: Breakable blocks and tricky AI enemies keep each level fresh and engaging.
- **⚡ Power-Ups**: Boost speed, extend bomb range, or grab extra bombs to gain the upper hand.
- **🧠 Survival Mechanics**: Avoid enemies and even your own bombs as you navigate each map.
- **🎮 Intuitive Controls**: Simple key controls for movement and bomb placement, with smooth level progression.
